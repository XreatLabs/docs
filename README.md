# XreatLabs Documentation

Welcome to the official documentation for XreatLabs - a developer studio creating innovative solutions for Minecraft servers, infrastructure systems, and developer tools.

XreatLabs builds design-forward systems for ambitious communities, platforms, and products. We specialize in:

- **Minecraft Plugins** - Performance optimization, authentication, and Discord integration
- **Infrastructure Tools** - Scalable backend systems for modern applications
- **Developer Experience** - Tools and documentation for better development workflows

## Local Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) to preview your documentation changes locally:

```bash
npm i -g mint
```

Run the following command at the root of your documentation:

```bash
mint dev
```

View your local preview at `http://localhost:3000`.

## AI-Assisted Development

Set up your AI coding tool to work with our documentation:

```bash
npx skills add https://docs.xreatlabs.space
```

This installs our documentation skill for AI tools like Claude Code, Cursor, and Windsurf.
