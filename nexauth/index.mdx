# NexAuth

<FeatureHero 
  title="Enterprise-Grade Authentication for Minecraft Servers"

## Security First

<Card
  title="Enterprise Security Features"
  icon="lock"
  href="/nexauth/features/security"
>

NexAuth implements industry-standard security practices to protect player accounts:

- **Secure Password Hashing** - bcrypt with adaptive work factor
- **Brute Force Protection** - Rate limiting and account lockouts
- **Session Management** - Configurable sessions with IP validation
- **Encryption at Rest** - Sensitive data encrypted in database
- **SQL Injection Prevention** - Parameterized database queries
- **Comprehensive Logging** - Authentication event audit trail

</Card>
